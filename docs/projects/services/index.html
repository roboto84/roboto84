Currently, I enjoy working with network applications which have an impact on my personal life. These network apps can run on the internet or on a LAN. Each setup has its pros and cons, but the apps can be configured for either. Currently, I host demos of some apps on the web, but I run the ones I use heavily on my local Fedora LAN server.

Air

A GPS location specific weather and air quality tracking system. Air uses Bind as a web UI, but can also run as a Wh00t chat bot, or simply a data collecting service which dumps weather data into an SQLite DB. The goal of Air is to help people which are sensitive to weather changes (due to allergies, migraines, ear aches, joint pain, etc) track weather and air quality patterns to see how the weather impacts their issues.

air_core

Core lib used to standardize certain concerns on multiple air projects.

air_collect

Weather data collection service which saves data to CSVs and SQLite DB.

air_bot

Wh00t chat bot that uses air to look up current weather information.

Lexicon

A personalized English dictionary made up of the words you curate. Lexicon uses Bind as a web UI, but can also run as a Wh00t chat bot, or as a terminal app. Lexicon helps you learn the words you do not know by looking up the definitions for the unknown words, and saving them in an SQLite DB for later study. A history of the words you have looked up and a "word of the day" feature are implemented.

lexicon

Core lib used by multiple lexicon projects for standardization, which also serves as a CLI.

lexicon_bot

Wh00t chat bot that uses lexicon for chat word dictionary look up capabilities.

kindle_word_collector

Program which transfers words looked up on a kindle dictionary to lexicon's SQLite DB

Arcadia

A website bookmarking system that allows you to relate websites with tags.

arcadia

Core lib used by multiple arcadia projects.

arcadia_bot

Wh00t chat bot that supplies arcadia bookmarking interface.

Gopher

Server stats tracking bot with support for Wh00t chat server.

gopher_bot

Wh00t chat bot that gives you system information on wherever gopher_bot is running.
